# App Icons

Bitte platziere hier die App-Icons:

- `icon-192x192.png` - 192x192 Pixel Icon
- `icon-512x512.png` - 512x512 Pixel Icon

## Icon-Anforderungen

- Format: PNG
- Hintergrund: Transparent oder mit Markenfarbe (#10B981 grün oder #7C3AED lila)
- Design: Einfach und klar erkennbar, idealerweise Mikrofon-Symbol
- Zweck: "any maskable" - sollte sowohl als normales als auch als maskiertes Icon funktionieren

## Temporäre Lösung

Bis echte Icons erstellt sind, kannst du Platzhalter verwenden oder Tools wie:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Canva oder Figma für eigene Designs

## Markenfarben (Arkonballon)

- Primär: #10B981 (Grün)
- Sekundär: #7C3AED (Lila/Violett)
